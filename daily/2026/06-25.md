## Notes

- Dial 的加入意味着 Cira 已不再局限于服务端场景，而是具备了真正的双向通信能力
- Stream 的生命周期应独立于普通 Request/Response 模型，其本质更接近长期存在的消息通道
- 匿名嵌入虽然方便，但容易暴露超出预期的接口，组合通常更加可控
- 命名往往决定抽象质量，client、peer、conn 等概念需要严格区分语义边界
- README 不只是文档，更是框架设计理念与使用方式的第一入口
- 示例代码是检验 API 是否自然的重要标准，很多设计问题会在示例阶段暴露出来
